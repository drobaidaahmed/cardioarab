🏥 الفحوصات الأساسية

  • ECG: شرط أساسي دائماً
  • Echocardiography: عند الاشتباه
  • Stress Test: للرياضيين المتقدمين
  • Cardiac MRI: لحالات خاصة
  • Angiography: عند الحاجة

📊 تصنيف الإجهاد الرياضي

  • A (منخفض): Golf, Darts
  • B (متوسط): Walking, Cycling
  • C (عالي): Running, Swimming
  • D (جداً عالي): Football, Basketball
  • الضغط النفسي: يؤثر أيضاً

💊 الأدوية الشائعة

  • Beta-Blockers: قد تقلل الأداء
  • ACEi/ARBs: آمنة عموماً
  • Statins: موصى بها
  • Diuretics: محظورة في المنافسات
  • Stimulants: محظورة كلياً

🚨 علامات التحذير الخطيرة

  • ألم صدري: توقف فوراً
  • ضيق التنفس الشديد: إنذار حقيقي
  • الإغماء أو الدوخة: تقييم فوري
  • الخفقان: قد يكون خطيراً
  • الشعور بالموت: قم بالإنعاش

👶 الفئات الخاصة

  • الأطفال: تقييم وراثي دقيق
  • الحمل: نشاط معتدل فقط
  • كبار السن: اختبارات شاملة
  • السكري: مراقبة مكثفة
  • ارتفاع الضغط: سيطرة أولاً

📋 الموافقات النهائية

  • Class I: واضح للموافقة
  • Class IIa: غالباً آمن
  • Class IIb: يحتاج دراسة أكثر
  • Class III: محظور أو خطر
  • عاد الفحص دورياً: 1-2 سنة
